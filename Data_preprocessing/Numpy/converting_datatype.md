# 데이터 형변환
- 데이터들끼리 형태를 맞춰줌
- 데이터가 어떤 타입인지 알고, 왜 바꾸는지 목적성을 가진 후 반환하기

1️⃣ np.data타입
```
data = [1.1 , 2, 3]
a = np.float64(data) 
print(a) # [1.1 2.  3. ] 실수형태로 변형
print(a.dtype)
```

2️⃣ astype 타입 바꾸기 / =(할당)
```
a= a.astype(np.int64) 
print(a)
print(a.dtype)
```
---------------------------------------
```
a = np.uint16(0)

a = a -1 
print(a) # -1
```
파이썬에서 자체 계산하여 음수로 표현
```
a = np.uint16(-1)
print(a) # 65535
```
직접 넣었을때, -1은 uint의 범위(0 ~ 65,535)에 들어가지 않기 때문에 범위의 마지막 값으로 출력됨
