## 집합(set)
중복 안됨,  순서 없음

1. 교집합
```
a = {1, 2, 3}
b = {2, 3, 4}

print(a & b)  # {2, 3}
print(a.intersection(b))  # {2, 3}
```

2. 합집합
```
a = {1, 2, 3}
b = {2, 3, 4}

print(a | b)  # {1, 2, 3, 4}
print(a.union(b))  # {1, 2, 3, 4}
```
3. 차집합
```
a = {1, 2, 3}
b = {2, 3, 4}

print(a | b)  # {1, 2, 3, 4}
print(a.union(b))  # {1, 2, 3, 4}
```
