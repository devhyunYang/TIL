## 정규 표현식?
- 불규칙해 보이는 텍스트 중 특정 패턴을 찾을 때
- 복잡한 문자열을 처리할 때

```
import re
```

|정규표현식|설명|예시|
|------|---|---|
|\||또는| A\|B |
|()|그룹|(a|b)|
|[]|문자 셋 or 괄호 한 모든 문자|[a-z]|
|[^]|[]와 반대, 괄호 안 문자 제외한 다른 것 다 |[^a-z]|
|(?:)|그룹 집합을 관리되지 않는다|(y)(?:x)|
|.|문자 개수 당 점 1개 |p...|
|^|문자열의 시작|^short|
|$|문자열의 끝|short$|
|\*|0개 이상|y*|
|?|0개 아니면 1|y?|
|+|1개 이상|y+|
